## 📦 Files Created

### 1. **src/train_model.py** (265 lines)
**Purpose**: Train machine learning models

**Features**:
- Loads dataset from `res/` folder
- Trains 3 models:
  - Decision Tree Classifier
  - Random Forest Classifier
  - Support Vector Machine (SVM)
- Displays accuracy for each model
- Saves trained models using joblib
- Encodes diseases to numeric labels
- Splits data 80-20 for training/testing

**Usage**: `python src/train_model.py`

---

### 2. **src/predictor.py** (190 lines)
**Purpose**: Make disease predictions

**Key Classes**:
- `DiseasePredictor`: Handles predictions

**Methods**:
- `load_models()`: Load trained models
- `predict_disease()`: Make predictions with confidence scores
- `get_all_symptoms()`: Get list of available symptoms
- `get_all_diseases()`: Get list of possible diseases
- `_normalize_scores()`: Convert SVM scores to probabilities

**Features**:
- Loads pre-trained models
- Creates feature vectors from symptoms
- Returns top 3 predictions with probabilities
- Handles all 3 models (Decision Tree, Random Forest, SVM)

---

### 3. **src/personal_health.py** (250 lines)
**Purpose**: Manage personal health profiles

**Key Classes**:
- `PersonalHealthProfile`: Manages user profiles

**Methods**:
- `create_profile()`: Create new user profile
- `save_profile()`: Save profile to JSON
- `load_profile()`: Load existing profile
- `calculate_risk_score()`: Calculate disease risk
- `add_prediction_to_history()`: Track predictions
- `get_prediction_history()`: Retrieve past predictions
- `validate_profile()`: Validate user input

**Features**:
- Create/load user profiles
- Store name, age, previous diseases
- Calculate risk scores (0-100)
- Track prediction history
- JSON-based storage

---

### 4. **src/gui_app.py** (600+ lines)
**Purpose**: Tkinter-based GUI application

**Key Classes**:
- `AIHealthcareGUI`: Main GUI application

**Screens**:
1. **Home Screen**
   - General Disease Prediction button
   - Personal Healthcare Prediction button
   - Exit button

2. **General Disease Prediction Screen**
   - Scrollable symptom checklist
   - Clear All button
   - Predict Disease button
   - Results display (Top 3 with probabilities)

3. **Personal Healthcare Prediction Screen**
   - Load existing profile or create new one
   - Profile creation form (name, age, diseases)
   - Symptom selection
   - Results with risk level (LOW/MEDIUM/HIGH)
   - Prediction history

**Features**:
- Responsive Tkinter interface
- Professional color scheme
- Scrollable symptom lists
- Dynamic screen management
- Error handling with message boxes
- Smooth navigation between screens
